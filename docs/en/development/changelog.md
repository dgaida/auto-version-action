# Changelog Workflow

We use `git-cliff` to automatically generate our changelog from conventional commits.

## Configuration

The configuration is located in `cliff.toml`. It groups commits into categories like "Features", "Bug Fixes", and "Documentation".

## Contributing

Please ensure your commit messages follow the Conventional Commits standard:

- `feat: ...` for new features
- `fix: ...` for bug fixes
- `docs: ...` for documentation changes
