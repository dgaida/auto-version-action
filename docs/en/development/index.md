# Development

Welcome to the development area. Here you can learn how to contribute to the project.

## Local Setup

1. Clone the repository.  
2. Create a virtual environment: `python -m venv venv`.  
3. Install dependencies: `pip install -e ".[test]"`.  

## Running Tests

We use `pytest` for our unit tests:

```bash
python -m pytest
```

## Code Style

We use `ruff` for linting and formatting. Please ensure your code meets the standards.
