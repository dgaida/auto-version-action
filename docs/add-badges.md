# Add Badges

Automatically detects features of your repository and adds status badges to your \`README.md\`.

## Detected Features

- Version (from \`pyproject.toml\` or \`package.json\`)
- Python Version
- MIT License
- Codecov
- GitHub Workflows (lint, tests, codeql)
- Code Style (Black, Ruff)
- Documentation (GitHub Pages or \`docs/\` folder)
